01 / Introduccion

Descripcion General

Debuggea tu Antojo es una Progressive Web App (PWA) creada como companion app oficial de Maruchan para el evento Talent Land 2026 en Expo Santa Fe, CDMX.

La aplicacion permite a los asistentes del evento interactuar con la marca Maruchan a traves de cupones flash, retos, trivias, logros desbloqueables, un mapa interactivo del venue y notificaciones push en tiempo real.

Que es una PWA?

Una Progressive Web App funciona como una app nativa desde el navegador. No requiere descarga desde tiendas de aplicaciones. Se accede desde un enlace web y puede instalarse en la pantalla de inicio del telefono.

02 / Instalacion

Instalacion como PWA

La app no requiere descarga desde App Store ni Google Play. Se instala directamente desde el navegador del telefono.

Requisitos

Navegador Chrome (Android) o Safari (iOS) actualizado. Conexion a internet para el primer acceso.

Pasos para instalar:

1. Abrir el enlace

Abre el URL de la app en Chrome o Safari desde tu celular.

2. Agregar a pantalla de inicio

Android: Toca el menu (tres puntos) > "Agregar a pantalla de inicio".
iOS: Toca el boton de compartir > "Agregar a inicio".

3. Abrir como app

La app aparecera como un icono en tu pantalla de inicio. Se abre en pantalla completa sin barra de navegador.

03 / Permisos

Permisos Solicitados

La app solicita un unico permiso para funcionar de forma optima. Este permiso es opcional y puede ser activado o desactivado en cualquier momento.

🔔

Notificaciones Push

Opcional

Permite recibir alertas en tiempo real sobre cupones flash, nuevos retos y actualizaciones de estaciones Maruchan.

Pantalla de solicitud de permisos de notificaciones

Para que sirve cada alerta:

🎟️

Cupones Flash

Recibe alertas instantaneas cuando un nuevo cupon de Maruchan esta disponible. Los cupones tienen tiempo y usos limitados.

⚡

Nuevos Retos

Enterate cuando se publiquen nuevas trivias o desafios para ganar puntos y desbloquear logros.

📍

Estaciones Maruchan

Recibe actualizaciones sobre la disponibilidad de las estaciones Maruchan en el venue (abiertas, ocupadas, cerradas).

Puedes cambiar esto despues

Si decides saltar este paso, puedes activar las notificaciones en cualquier momento desde tu perfil dentro de la app.

Permisos que NO solicita la app:

📷

Camara

La app no accede a tu camara.

📍

Ubicacion / GPS

No se rastrea tu ubicacion.

👤

Contactos

No accede a tus contactos.

🎙️

Microfono

No accede a tu microfono.

06 / Cupones

Cupones Flash

15 / Tecnico

Datos Tecnicos

Informacion tecnica de la aplicacion para referencia del equipo.

Tipo de aplicacion

Progressive Web App (PWA) - No requiere descarga desde tiendas.

Tecnologias

Next.js 16 + TypeScript + Tailwind CSS v4 + Prisma + SQLite

Notificaciones

Web Push con VAPID keys via Service Worker

Almacenamiento

Sesion de usuario en localStorage. Datos de juego en base de datos del servidor.

Offline

Service Worker pre-cachea rutas principales para funcionamiento basico sin conexion.

Autenticacion

Basada en localStorage (userId). Sin contrasenas ni datos sensibles.

Privacidad

La app solo almacena: nickname, modo de juego, configuracion del avatar y estado de notificaciones. No recopila datos personales, ubicacion, ni informacion del dispositivo.
